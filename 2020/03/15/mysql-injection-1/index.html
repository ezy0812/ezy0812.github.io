mysql的目录结构

mysql 5.0以上的版本中，默认定义了information_schema数据库，其中包含有表schemata(数据库名)、tables、columns(列名)

表名 含有重要列名
schemata schema_name
tables table_schema, table_name
columns table_schema, table_name, column_name
