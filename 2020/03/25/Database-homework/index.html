Add a new attribute to the table created in the previous question using the alter command. The new attribute should have the domain time and be named attime

alter table q1 add attime TIME;

worksheet 2

List the names of all customers who have never placed an order

SELECT customerName, orderNumber FROM customers LEFT JOIN orders USING(customerNumber) WHERE orderNumber IS NULL;

List the names of all employees who work for Gerard Bondur (Hint: the “reportsTo” attribute in the “employees” table contains the employeeNumber of the person each employee works for).

SELECT e.employeeNumber, e.firstName, e.lastName FROM employees AS e JOIN employees AS m ON e.reportsTo=m.employeeNumber WHERE m.firstName='Gerard' AND m.lastName='Bondur';

The employee named George Vanauf has moved to the Paris office. Make a change to the database to record this information.

UPDATE employees SET officeCode=(SELECT officeCode FROM offices WHERE city='Paris') WHERE firstname='George' AND lastname='Vanauf';

worksheet 3

Write an SQL statement to display the detail of the payment with the highest amount

SELECT * FROM payments ORDER BY amount DESC limit 1;

How much money did the company earn from orders of Classic Car models in 2004?

SELECT SUM(quantityOrdered*priceEach) FROM orders INNER JOIN orderdetails USING(orderNumber) INNER JOIN products USING(productCode) WHERE orderDate LIKE '2004%' AND productLine='Classic Cars';

Name the customer(s), product and quantity ordered which represents the largest quantity of any item ordered?

SELECT customerName, productCode, quantityOrdered FROM orderdetails INNER JOIN orders USING(orderNumber) INNER JOIN customers USING(customerNumber) WHERE quantityOrdered=(SELECT MAX(quantityOrdered) FROM orderdetails);

List the number of products in each product line.

SELECT productLine, COUNT(*) FROM products GROUP BY productLine;

What is the average quantity of model ordered for each product line?

SELECT productLine, AVG(quantityOrdered) FROM products INNER JOIN orderdetails USING(productCode) GROUP BY productLine;

List those product lines, and the number of products they have, who have more than ten products in the line

SELECT productLine, COUNT(*) FROM products GROUP BY productLine HAVING COUNT(*)>10;

List the name and price of the 10 most expensive products (according to the MSRP price), in order

SELECT productName, MSRP FROM products ORDER BY MSRP DESC LIMIT 10;

Create a view, based on the products table, that only contains model ships. This can be named ‘ships’

CREATE VIEW ships AS SELECT * FROM products WHERE productLine='Ships';

List the customer name and the total amount paid by any customer who has paid more than 250000 in total.

SELECT customerName, SUM(amount) FROM payments INNER JOIN customers USING(customerNumber) GROUP BY customerNumber HAVING SUM(amount) > 250000;

Using a nested query, list the name and city of any customer that is based in a city that the company has an office in.

SELECT customerName, city FROM customers WHERE city = ANY(SELECT city FROM offices);

Or use join

SELECT customerName, city FROM customers INNER JOIN offices USING(city);
