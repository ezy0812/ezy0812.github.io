查询文件信息

  • st_size:文件的体积大小(bytes)，除以1024就是KB
  • st_atime:文件的最近访问时间
  • st_mtime:文件的最近修改时间
  • st_ctime:Windows下表示创建时间
  • st_birthtime:只在Mac,Linux下可用，表示创建时间

上面的时间都是类似1584868409这样的数字，这种时间叫做Unix时间戳，可以利用Python转换为正常日期时间

写入，读取文件

一种建议的写法，不管程序是否正常运行都会关闭文件，不需要再单独写close()

写文件

  • ‘w’表示写入，若文件不存在则会创建新文件，若存在，则会将文件原内容清空，然后写入内容
  • ‘a’表示追加，若件不存在则会创建新文件，若存在，则会在文件最后追加内容，而不清空原内容

创建临时文件及文件夹

  • ‘w+’表示写入及读取文件
  • seek(0)表示回到文件开头位置
  • 在程序运行完之后会自动删掉临时文件
