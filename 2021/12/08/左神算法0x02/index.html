0x01 判断回文

给定单链表头结点head，判断一个链表是否为回文

1->2->1 true

1->2->2->1 true

1.0

遍历单链表，压入栈，然后一个一个弹出，与单链表比对，额外空间复杂度O(n)

public static boolean isPalindrome1(Node head){
    Stack<Node> stack = new Stack<Node>();
    Node cur = head;
    while(cur!=null){
        stack.push(cur);
        cur = cur.next;
    }
    cur = head;
    while(cur!=null){
        if(stack.pop().value!=cur.value) return false;
        cur = cur.next;
    }
    return true;
}

2.0

找到链表中间，然后开始压入栈，这样只需要一半空间，O(n/2)

那么怎么找到中间位置，利用快慢指针，即从头建立两个指针，一个每次走2步，一个每次走1步，当快指针走到结尾，那么满指针就走到中间了

public static boolean isPalindrome2(Node head){
    Stack<Node> stack = new Stack<>();
    if(head == null || head.next == null) return true;
    Node slow = head;
    Node fast = head;
    while(fast.next!=null&&fast.next.next!=null){
        slow = slow.next;
        fast = fast.next.next;
    }
    while(slow.next!=null){
        stack.push(slow.next);
        slow = slow.next;
    }
    slow = head;
    while(!stack.isEmpty()){
        if(stack.pop().value!=slow.value) return false;
        slow = slow.next;
    }
    return true;
}

3.0

同样利用快慢指针，慢指针到达中间时，将中间的这个节点的next指到null，然后原来的next的next指到这个中间的节点，依次逆序过来，完成之后，开始一一比对，这样的空间复杂度O(1)，不要忘记最后将右边一半恢复

public static boolean isPalindrome3(Node head){
    if(head == null || head.next == null) return true;
    Node slow = head;
    Node fast = head;
    while(fast.next!=null&&fast.next.next!=null){
        slow = slow.next;
        fast = fast.next.next;
    }
    // 逆序
    Node par = null;
    Node curNext;
    while (slow != null) {
        curNext = slow.next;
        slow.next = par;
        par = slow;
        slow = curNext;
    }
    Node cur = head;
    Node tail = par;
    boolean res = true;
    // 比较值
    while(cur!=null&&par!=null){
        if (cur.value != par.value) {
            res = false;
            break;
        }
        par = par.next;
        cur = cur.next;
    }
    // 调整回来
    par = null;
    while (tail != null) {
        curNext = tail.next;
        tail.next = par;
        par = tail;
        tail = curNext;
    }
    return res;
}

0x02 链表的荷兰国旗问题

将单链表按某值划分为左边小，中间相等，右边大的形式

1.0

将链表放到辅助Node数组，于是变成了荷兰国旗问题，空间复杂度O(n)

public static Node lessEqualMore1(Node head, int value){
    if(head == null || head.next == null) return head;
    int len = 0;
    Node cur = head;
    while(cur!=null){
        len++;
        cur = cur.next;
    }
    Node[] nodes = new Node[len];
    cur=head;
    for(int i=0;i<len;i++){
        nodes[i] = cur;
        cur = cur.next;
    }
    int lessArea = -1;
    int moreArea = len;
    for(int i=0;i<moreArea;){
        if(nodes[i].value < value) swap(nodes, i++, ++lessArea);
        else if(nodes[i].value > value) swap(nodes, i, --moreArea);
        else i++;
    }
    for(int i=1;i<len;i++){
        nodes[i-1].next = nodes[i];
    }
    nodes[len-1].next = null;
    return nodes[0];
}

public static void swap(Node[] arr, int a, int b){
    int temp = arr[a].value;
    arr[a].value = arr[b].value;
    arr[b].value = temp;
}

2.0

只需要六个变量，小于区域的头节点LH，尾节点LT，等于区域的头节点EH，尾节点ET，大于区域的头节点MH，尾节点MT，遍历单链表，第一次遇到小于的，就将LH和LT都指向它，第二次遇到小于的，就将LT的next指向它，然后LT指向它，其它区域同样，遍历完成之后，将SH、ST、EH、ET、MH、MT依次连接，同时注意空指针的情况，即有可能不存在小于区域，或者其它区域，导致没有next节点，如果不处理，则会报出异常

public static Node lessEqualMore2(Node head, int value){
    Node LH = null;
    Node LT = null;
    Node EH = null;
    Node ET = null;
    Node MH = null;
    Node MT = null;
    Node cur = head;
    while(cur!=null){
        Node temp = cur.next;
        cur.next = null;
        if(cur.value < value){
            if(LH == null) LH = cur;
            else LT.next = cur;
            LT = cur;
        }
        else if(cur.value == value){
            if(EH == null) EH = cur;
            else ET.next = cur;
            ET = cur;
        }
        else{
            if(MH == null) MH = cur;
            else MT.next = cur;
            MT = cur;
        }
        cur = temp;
    }
    if(LT!=null) {
        LT.next = EH;
        ET = (ET == null)?LT:ET;
    }
    if(ET!=null) ET.next = MH;
    return (LH!=null)?LH:(EH!=null)?EH:MH;
}

0x03 复制链表

复制含有随机指针节点的链表

public static class Node {
    public int value;
    public Node next;
    public Node rand;

    public Node(int data) {
        this.value = data;
    }
}

1.0

使用哈希表，key为旧链表的节点，value为它的克隆节点，完成哈希表建立之后，遍历旧链表，节点的next即为节点对应value的next，rand同理

public static Node CopyListWithRandom1(Node head){
    HashMap<Node, Node> hash = new HashMap<>();
    Node cur = head;
    while(cur!=null){
        hash.put(cur,new Node(cur.value));
        cur = cur.next;
    }
    cur = head;
    while(cur!=null){
        hash.get(cur).next = hash.get(cur.next);
        hash.get(cur).rand = hash.get(cur.rand);
        cur = cur.next;
    }
    return hash.get(head);
}

2.0

1->2->3

遍历链表，每个旧节点的next指向为该节点的克隆节点，结果即为1->1’->2->2’->3->3’

对于rand，如果1的rand指向3，那么1’的rand顺理成章指向1的rand的next，其它节点同理

对于next，只需要将该链表拆开即可完成

public static Node CopyListWithRandom2(Node head){
    if(head == null) return null;
    Node cur = head;
    while(cur!=null){
        Node curNext = cur.next;
        cur.next = new Node(cur.value);
        cur.next.next = curNext;
        cur = curNext;
    }
    cur = head;
    while(cur!=null){
        cur.next.rand = cur.rand == null?null:cur.rand.next;
        cur = cur.next.next;
    }
    Node res = head.next;
    cur = head;
    while(cur!=null){
        Node oldNext = cur.next.next;
        cur.next.next = oldNext == null ? null : oldNext.next;
        cur.next = oldNext;
        cur = oldNext;
    }
    return res;
}

0x04 两个单链表相交

两个可能有环也可能无环的单链表，如果相交（即内存地址相同），则返回相交的第一个节点，否则返回null

一）首先需要判断两个链表是否有环

getLoopNode(Node head)有环则返回入环节点（即第一个进入环中的节点），否则返回null

使用快慢指针

a）无环，则快指针一定会到达null

b）有环，则快指针一定会与慢指针相遇，相遇之后则停，让快指针跑到头节点，慢指针停在这个相遇点，然后两者都开始每次走一步，再次相遇的点则为入环节点（证明比较麻烦，此处不证明）

public static Node getLoopNode(Node head){
    if(head == null || head.next == null || head.next.next == null) return null;
    // 不让快慢指针都从头开始的原因是需要进入下面的while循环
    Node slow = head.next;
    Node fast = head.next.next;
    while(slow!=fast){
        if(fast.next == null || fast.next.next == null) return null;
        slow = slow.next;
        fast = fast.next.next;
    }
    fast = head;
    while(fast!=slow){
        fast = fast.next;
        slow = slow.next;
    }
    return fast;
}

二）分情况讨论

a）两个都无环的链表（loop1 == null && loop2 == null）分别遍历两个链表，得到两个链表的长度差值和尾部节点，首先比较尾部节点是否为同一个（如果同一个则一定相交，下面开始找相交的第一个节点，反之，不是同一个的话，一定不相交），然后从长的开始遍历差值步，然后与短的链表一起继续向后遍历，遍历到相同内存地址，则为相交的第一个节点

public static Node noLoop(Node head1, Node head2){
    // 只使用一个变量n来获得两个链表的差值，省空间大师
    int n = 0;
    Node cur1 = head1;
    while(cur1.next != null){
        cur1 = cur1.next;
        n++;
    }
    Node cur2 = head2;
    while(cur2.next !=null){
        cur2 = cur2.next;
        n--;
    }
    // 执行完上面两个while之后，cur1和cur2即为两个链表的尾部节点
    if(cur1 == cur2){
        // 现在让cur1为长的链表，cur2为短的链表
        cur1 = n > 0 ? head1 : head2;
        cur2 = cur1 == head1 ? head2 : head1;
        n = Math.abs(n);
        while(n!=0){
            cur1 = cur1.next;
            n--;
        }
        while(cur1 != cur2){
            cur1 = cur1.next;
            cur2 = cur2.next;
        }
        return cur1;
    }
    return null;
}

b）两个都有环的链表（loop1 != null && loop2 != null）总共就分3种情况，即为下图

图中第二种情况即为loop1 == loop2，寻找这种情况的第一个相交节点则与a）情况两个都无环链表类似，将loop1或者loop2节点当做两个链表的尾部节点即可采用完全相同的代码

那么如果两者不相等，则从loop1开始向后遍历，如果相交，则一定会遍历到loop2，返回loop1或者loop2都满足题意，否则一定不相交

public static Node bothLoop(Node head1, Node head2, Node loop1, Node loop2){
    if(loop1 == loop2){
        int n = 0;
        Node cur1 = head1;
        while(cur1 != loop1){
            cur1 = cur1.next;
            n++;
        }
        Node cur2 = head2;
        while(cur2 != loop1){
            cur2 = cur2.next;
            n--;
        }
        // 执行完上面两个while之后，cur1和cur2即为两个链表的入环节点
        // 现在让cur1为长的链表，cur2为短的链表
        cur1 = n > 0 ? head1 : head2;
        cur2 = cur1 == head1 ? head2 : head1;
        n = Math.abs(n);
        while(n!=0){
            cur1 = cur1.next;
            n--;
        }
        while(cur1 != cur2){
            cur1 = cur1.next;
            cur2 = cur2.next;
        }
        return cur1;
    }
    else{
        Node cur = loop1.next;
        while(cur != loop1){
            if(cur == loop2) return cur;
            cur = cur.next;
        }
        return null;
    }
}

c）剩下的情况则是有一个链表为无环，另一个则为有环，这种情况一定不会相交

一定记住前提为单链表，即每个节点只有一个next指针，所以以上情况完全囊括了全部可能性

public static Node findFirstIntersecNode(Node head1, Node head2){
    if(head1 == null || head2 == null) return null;
    Node loop1 = getLoopNode(head1);
    Node loop2 = getLoopNode(head2);
    if(loop1 == null && loop2 == null) return noLoop(head1, head2);
    else if(loop1 != null && loop2 != null) return bothLoop(head1, head2, loop1, loop2);
    return null;
}

二叉树

遍历问题

public static class Node{
    public int value;
    public Node left;
    public Node right;
    
    public Node(int data){
        this.value = data;
    }
}
public static void f(Node head){
    if(head == null) return;
    // 第一次回到当前子树的根节点
    f(head.left);
    // 第二次回到当前子树的根节点
    f(head.right);
    // 第三次回到当前子树的根节点
}

显然，对于先序、中序、后序遍历的递归实现往对应的注释出添加打印语句即可

那么对于迭代怎么实现？

先序遍历

首先准备一个栈，先压入根节点，然后循环以下操作

1）弹出一个节点，即为cur

2）打印

3）先压入右节点，再压入左节点

public static void preTraversal(Node head){
    if(head != null) {
        Stack<Node> stack = new Stack<>();
        stack.push(head);
        while (!stack.isEmpty()) {
            Node cur = stack.pop();
            System.out.println(cur.value+" ");
            if (cur.right != null) stack.push(cur.right);
            if (cur.left != null) stack.push(cur.left);
        }
    }
}

后序遍历

由前面的先序遍历可知先压右后压左得出的结果为（根、左、右），那么先压左后压右则得出（根、右、左），所以将（根、右、左）放在另一个栈里，再弹出，则为左、右、根

首先准备两个栈，先压入根节点到栈A中，循环以下操作

1）弹出一个节点，即为cur

2）将cur压入栈B

3）先压入左节点，再压入右节点

打印栈B即可

public static void postTraversal(Node head){
    if(head != null) {
        Stack<Node> stackA = new Stack<>();
        Stack<Node> stackB = new Stack<>();
        stackA.push(head);
        while (!stackA.isEmpty()) {
            Node cur = stackA.pop();
            stackB.push(cur);
            if (cur.left != null) stackA.push(cur.left);
            if (cur.right != null) stackA.push(cur.right);
        }
        while(!stackB.isEmpty()) System.out.println(stackB.pop().value+" ");
    }
}

中序遍历

对于中序来说，入栈顺序则为左、右、根

1）将以该节点为根节点的树的左子节点全部入栈（即整棵树的左边界入栈）

2）弹出cur

3）打印，对cur右子树循环操作

public static void inTraversal(Node head){
    if(head != null) {
        Stack<Node> stack = new Stack<>();
        // 巧妙复用head变量简化代码
        while(!stack.isEmpty() || head != null){
            if(head != null){
                stack.push(head);
                head = head.left;
            }else{
                head = stack.pop();
                System.out.println(head.value+" ");
                head = head.right;
            }
        }
    }
}

宽度优先遍历

使用队列结构，首先压入根节点，然后循环以下操作

1）弹出

2）打印

3）先压入左节点，再压入右节点

public static void widthFirstTraversal(Node head){
    if(head != null){
        Queue<Node> queue = new LinkedList<>();
        queue.add(head);
        while(!queue.isEmpty()){
            Node cur = queue.poll();
            System.out.println(cur.value+" ");
            if (cur.left != null) queue.add(cur.left);
            if (cur.right != null) queue.add(cur.right);
        }
    }
}

0x01 求二叉树的宽度

1.0

采用哈希表，存储每一个节点对应在第几层，从根节点在第一层开始，按照宽度优先遍历的步骤，使用变量存储当前在第几层、当前层数已经遍历的节点数量、当前已知的最大宽度(max)

从第一层开始，将根节点入队，出队发现当前在第一层，出队的节点查表得知在第一层，所以开始累积第一层已经遍历的节点数量，为1，跟随宽度优先遍历，入队的同时，也将节点的宽度入表

到了第二层，查表发现出队的节点的层数与当前层数不一致，当前层数的节点遍历完毕，可以结算节点的数量，与max比较即可，当前的层数来到第二层，当前层数已经遍历的节点数量也刷新（这里注意，因为现在到了第二层，遍历完了第一个节点，所以刷新值为1而不是0），循环下去即可

public static int getMaxWidthOne(Node head){
    if(head != null){
        Queue<Node> queue = new LinkedList<>();
        HashMap<Node, Integer> hm = new HashMap<>();
        queue.add(head);
        hm.put(head, 1);
        int curLevel = 1;
        int curLevelNodes = 0;
        int max = Integer.MIN_VALUE;
        while(!queue.isEmpty()){
            Node cur = queue.poll();
            int curNodeLevel = hm.get(cur);
            if(curNodeLevel == curLevel) curLevelNodes++;
            else{
                max = Math.max(max, curLevelNodes);
                curLevel++;
                curLevelNodes = 1;
            }
            if (cur.left != null) {
                hm.put(cur.left, curNodeLevel+1);
                queue.add(cur.left);
            }
            if (cur.right != null) {
                hm.put(cur.right, curNodeLevel+1);
                queue.add(cur.right);
            }
        }
        return max;
    }
    return 0;
}

2.0

不使用哈希表，只使用队列，额外变量记录当前层的最后一个节点、下一层的最后一个节点、当前层的节点数量、当前已知的最大宽度(max)

根节点入队，出队，当前层的最后一个节点赋为根节点，下一层的最后一个节点赋为空，然后开始左右节点入队，保持下一层的最后一个节点总是为最后入队的节点（即队尾的节点），当到达当前层的最后一个节点时，结算，进行下一轮

public static int getMaxWidthTwo(Node head){
    if(head != null){
        Queue<Node> queue = new LinkedList<>();
        queue.add(head);
        int max = Integer.MIN_VALUE;
        int curLevelNodes = 1;
        Node curEnd = queue.peek();
        Node nextEnd = null;
        while(!queue.isEmpty()){
            Node cur = queue.poll();
            if (cur.left != null) {
                queue.add(cur.left);
                nextEnd = cur.left;
            }
            if (cur.right != null) {
                queue.add(cur.right);
                nextEnd = cur.right;
            }
            if(cur == curEnd){
                curEnd = nextEnd;
                nextEnd = null;
                max = Math.max(max, curLevelNodes);
                curLevelNodes = 1;
            }else{
                curLevelNodes++;
            }
        }
        return max;
    }
    return 0;
}

0x02 判断搜索二叉树

搜索二叉树即左子树的值都比根节点小，右子树的值都比根节点大，一棵经典的搜索二叉树是没有重复值的

使用中序遍历即可，中序遍历搜索二叉树即为升序，所以如果一旦遇到逆序的，则不是搜索二叉树，需要在中序遍历的打印处修改为判断结构即可

1.0 递归的改写

需要一个全局变量记录遍历到的前一个值，然后与遍历的值比较即可

public static int pre = Integer.MIN_VALUE;
public static boolean isBSTOne(Node head){
    if(head == null) return true;
    boolean isBST = isBSTOne(head.left);
    if(!isBST) return false;
    if(head.value <= pre) return false;
    pre = head.value;
    return isBSTOne(head.right);
}

2.0 迭代的改写

public static boolean isBSTTwo(Node head){
    if(head != null) {
        Stack<Node> stack = new Stack<>();
        int pre = Integer.MIN_VALUE;
        while(!stack.isEmpty() || head != null){
            if(head != null){
                stack.push(head);
                head = head.left;
            }else{
                head = stack.pop();
                if(head.value <= pre) return false;
                else pre = head.value;
                head = head.right;
            }
        }
    }
    return true;
}

0x03 判断完全二叉树

即要么所有层都是满的，要么最后一层即使不满也是从左到右依次占满

显然可以使用宽度遍历，遍历过程中进入如下判断

1）如果有右无左，false

2）过了1，说明不会存在有右无左，则现在如果出现有一个子节点为null，则后面的节点必须都为null，否则false，如何判断后面的节点，需要设立一个布尔变量标志是否遇到左右两个子节点不双全的情况

public static boolean isCBT(Node head){
    if(head != null){
        Queue<Node> queue = new LinkedList<>();
        queue.add(head);
        boolean leaf = false;
        while(!queue.isEmpty()){
            Node cur = queue.poll();
            Node l = cur.left;
            Node r = cur.right;
            if(
                    (leaf &&(l!=null || r!=null)) || (l == null && r != null)
            ) return false;
            if(l == null || r == null) leaf = true;
            if (l != null) queue.add(l);
            if (r != null) queue.add(r);
        }
    }
    return true;
}

0x04 二叉树的递归套路（树型DP）

左树提供什么信息，右树提供什么信息，然后怎么整合返回给上一层的递归

使用前提： 如果题目求解目标是S规则，则求解流程可以定成以每一个节点为头节点的子树在S规则下的每一个答案，并且最终答案一定在其中

树形dp套路

1）以某个节点X为头节点的子树中，分析答案有哪些可能性，并且这种分析是以X的左子树、X的右子树和X整棵树的角度来考虑可能性的

2）根据第一步的可能性分析，列出所有需要的信息

3）合并第二步的信息，对左树和右树提出同样的要求，并写出需要返回的信息结构

4）设计递归函数，递归函数是处理以X为头节点的情况下的答案。 包括设计递归的basecase，默认直接得到左树和右树的所有信息，以及把可能性做整合，并且要返回第三步的信息结构这四个小步骤

判断平衡二叉树

指左子树与右子树高度之差不超过1

递归过程中，需要左子树和右子树提供的信息为1）是否为平衡二叉树，2）该子树的高度

那么整棵子树也需要给上一层递归提供同样的上述两个信息，书写递归即可

public static class returnType{
    public boolean isBalanced;
    public int height;

    public returnType(boolean isBalanced, int height){
        this.isBalanced = isBalanced;
        this.height = height;
    }
}

public static boolean isBalancedTree(Node head){
    return process(head).isBalanced;
}

public static returnType process(Node head){
    if(head == null) return new returnType(true, 0);
    returnType lreturn = process(head.left);
    returnType rreturn = process(head.right);

    boolean isBalanced = (lreturn.isBalanced) && (rreturn.isBalanced) && (Math.abs((lreturn.height) - (rreturn.height)) <= 1);
    int height = Math.max(lreturn.height, rreturn.height) + 1;
    return new returnType(isBalanced, height);
}

判断搜索二叉树

这里需要左子树提供1）是否为搜索二叉树，2）该树最大值（需要小于根节点），右子树提供1）是否为搜索二叉树，2）该树最小值（需要大于根节点）

然而，我们需要递归返回同样的信息，现在出现一个需要最大值，一个需要最小值，所以我们选择将两者都返回

public static class returnData{
    public boolean isBST;
    public int max;
    public int min;

    public returnData(boolean isBST, int max, int min){
        this.isBST = isBST;
        this.max = max;
        this.min = min;
    }
}

public static boolean isBST(Node head){
    return process(head).isBST;
}

public static returnData process(Node head){
    if(head == null) return null;
    returnData lData = process(head.left);
    returnData rData = process(head.right);
    int min = head.value;
    int max = head.value;
    if(lData != null){
        min = Math.min(lData.min, min);
        max = Math.max(lData.max, max);
    }
    if(rData != null){
        min = Math.min(rData.min, min);
        max = Math.max(rData.max, max);
    }

    boolean isBST = true;
    if(lData != null) isBST = lData.isBST && (lData.max < head.value);
    if(rData != null) isBST = isBST && rData.isBST && (rData.min > head.value);
    return new returnData(isBST, max, min);
}

    if(head == null) return new returnData(0, 0);
    returnData lData = process(head.left);
    returnData rData = process(head.right);
    return new returnData(Math.max(lData.height, rData.height) + 1, lData.nodes + rData.nodes + 1);

二叉树节点间最大距离

从二叉树的节点a出发，可以向上或者向下走，但沿途的节点只能经过一次，到达节点b时路径上的节点个数叫作a到b的距离，那么二叉树任何两个节点之间都有距离，求整棵树上的最 大距离

想得到以x为根节点的树的最大距离，该距离分为两种情况

1）距离经过了x（x参与），那么想得到最大距离就必须是左子树高度+1+右子树高度

2）距离不经过x（x不参与），要么是左子树的最大距离，要么是右子树的最大距离

那么我们需要子树提供的信息就是：

1）子树的最大距离，2）子树的高度

public static class returnData{
    public int height;
    public int distance;
    public returnData(int h, int d){
        height = h;
        distance = d;
    }
}

public static int maxDistance(Node head){
    return process(head).distance;
}

public static returnData process(Node node){
    if(node == null) return new returnData(0,0);
    returnData left = process(node.left);
    returnData right = process(node.right);

    int dis = Math.max(left.distance, Math.max(right.distance, left.height + right.height + 1));
    int hei = Math.max(left.height, right.height) + 1;
    return new returnData(hei, dis);
}

最大快乐值（多叉树）

员工信息的定义如下: 

public static class Employee { 
    public int happy; // 这名员工可以带来的快乐值 
    List<Employee> subordinates; // 这名员工有哪些直接下级 
}

公司的每个员工都符合 Employee 类的描述。整个公司的人员结构可以看作是一棵标准的、 没有环的多叉树。树的头节点是公司唯一的老板。除老板之外的每个员工都有唯一的直接上级。 叶节点是没有任何下属的基层员工(subordinates列表为空)，除基层员工外，每个员工都有一个或多个直接下级。

这个公司现在要办party，你可以决定哪些员工来，哪些员工不来。但是要遵循如下规则

1）如果某个员工来了，那么这个员工的所有直接下级都不能来

2）派对的整体快乐值是所有到场员工快乐值的累加

3）你的目标是让派对的整体快乐值尽量大

给定一棵多叉树的头节点boss，请返回派对的最大快乐值

想得到以x为根节点的树的最大快乐值，同样分为两种情况：

1）x参与，快乐值为x.happy + {所有直接下级不来的整颗子树的最大快乐值}

2）x不参与，快乐值为0 + {所有直接下级max(直接下级来的整颗子树的最大值，直接下级不来的整颗子树的最大值)}

public static class returnData{
    public int present;
    public int absent;
    public returnData(int p, int a){
        present = p;
        absent = a;
    }
}

public static int maxHappy(Employee boss){
        returnData bos = process(boss);
        return Math.max(bos.present, bos.absent);
}

public static returnData process(Employee employee){
    List<Employee> list = employee.subordinates;
    if(list.isEmpty()) return new returnData(employee.happy, 0);
    int p = employee.happy;
    int a = 0;
    for(Employee each:list){
        returnData emplo = process(each);
        p += emplo.absent;
        a += Math.max(emplo.present, emplo.absent);
    }
    return new returnData(p, a);
}

最低公共祖先节点

给定两个二叉树的节点node1和node2，找到他们的最低公共祖先节点

如果有公共祖先，只有下面两种情况

1）node1和node2，其中一个是另一个的祖先

2）都往上找，找到一个节点为公共祖先

使用递归遍历整棵树（代码不好理解）

base case：当前节点为空/node1/node2，返回当前节点

递归左右子树，然后如果两侧的结果都不为空的话，说明是情况 2），将当前节点往上层返回即可，否则说明至少有一侧为空，那么返回不为空的那一侧即可（两侧都为空，返回空）

public static Node lowestCommonAncestor(Node head, Node o1, Node o2){
    if(head == null || head == o1 || head == o2) return head;
    Node left = lowestCommonAncestor(head.left, o1, o2);
    Node right = lowestCommonAncestor(head.right, o1, o2);
    if(left != null && right != null) return head;
    return left != null ? left : right;
}

后继节点

即中序遍历中的下一个节点

节点类型添加了一个指向父节点的指针，根节点的父节点为null

public class Node {
    public int value;
    public Node left;
    public Node right;
    public Node parent;
    public Node(int val) {
        value = val;
    }
}

显然题意并不想让我们中序遍历整棵树，从而得出后继节点，这样的复杂度是O（n），但是有了父节点的帮助，我们可以优化到O（k），k为该节点与后继节点的高度差

这就需要我们理解树的结构，节点分为两种情况

1）该节点有右子树 - 返回右子树的最左边的子节点

2）无右子树 - 向上找，如果不是父亲的左孩子，继续向上，找到第一个是父亲的左孩子为止，那么该父亲即为后继节点

public static Node successorNode(Node head, Node node){
    if(head == null || node == null) return null;

    Node cur = node.right;
    if(cur != null){
        while(cur.left != null) cur = cur.left;
        return cur;
    }

    Node par = node.parent;
    cur = node;
    while(par != null && par.left != cur){
        cur = par;
        par = par.parent;
    }
    return par;
}

Morris遍历

一种遍历二叉树的方式，并且时间复杂度O(N)，额外空间复杂度O(1)

通过利用原树中大量空闲指针的方式，达到节省空间的目的

Morris遍历细节

假设来到当前节点cur，开始时cur来到头节点位置

1）如果cur没有左孩子，cur向右移动(cur = cur.right)

2）如果cur有左孩子，找到左子树上最右的节点mostRight：

  • 如果mostRight的右指针指向空，让其指向cur， 然后cur向左移动(cur = cur.left)

  • 如果mostRight的右指针指向cur，让其指向null， 然后cur向右移动(cur = cur.right)

3）cur为空时遍历停止

有左子树的节点可以回到该节点两次，没有左子树的节点只能到达1次

有左子树的节点可以根据左子树的最右节点的指向来判断第几次到达该节点，指向自己就是第二次

public static void morris(Node head){
    if(head == null) return;
    Node cur = head;
    Node mostRight;
    while(cur != null){
        mostRight = cur.left; // 只是先初始化为左子树的根节点
        // 是否有左孩子
        if(mostRight != null){
            // 找到最右节点
            while(mostRight.right != null && mostRight.right != cur) mostRight = mostRight.right;
            if(mostRight.right == null){ // 说明第一次来到cur节点，就让现在的最右节点的右指针指向cur
                mostRight.right = cur;
                cur = cur.left;
                continue;
            }else mostRight.right = null;
        }
        cur = cur.right;
    }
}

public static Node reverse(Node head){

    Node cur = head.right;
    Node temp;
    par.right = null;
    while(cur != null) {
        temp = par;
        par = cur;
        cur = cur.right;
        par.right = temp;
    }
    return par;
}

总结

使用树形DP：方法必须做第三次信息的强整合（一个节点必须要完左子树的信息，要完右子树的信息，然后回到该节点，做整合），必须用树形DP

使用Morris遍历：不需要强整合，可以用树形DP，但是Morris遍历是最优解（因为Morris遍历不能第三次回到某个节点）

        Integer weight = matrix[i][0];
        Integer from = matrix[i][1];
        Integer to = matrix[i][2];
        if (!graph.nodes.containsKey(from)) {
            graph.nodes.put(from, new Node(from));
        }
        if (!graph.nodes.containsKey(to)) {
            graph.nodes.put(to, new Node(to));
        }
        Node fromNode = graph.nodes.get(from);
        Node toNode = graph.nodes.get(to);
        Edge newEdge = new Edge(weight, fromNode, toNode);
        fromNode.nexts.add(toNode);
        fromNode.out++;
        toNode.in++;
        fromNode.edges.add(newEdge);
        graph.edges.add(newEdge);
    }
    return graph;
}

宽度优先遍历

建立一个队列和一个哈希set，从给定节点开始，入队，出队，打印，入队的同时注意在哈希set里也将其加入，为了防止A指向C，C又指向A，死循环，可以理解为节点进入了set，就相当于已经注册了

public static void BFS(Node n){
    if(n == null) return;
    Queue<Node> queue = new LinkedList<>();
    HashSet<Node> set = new HashSet<>();
    queue.add(n);
    set.add(n);
    while(!queue.isEmpty()){
        Node cur = queue.poll();
        System.out.println(cur.value);
        for(Node next:cur.nexts){
            if(!set.contains(next)){
                set.add(next);
                queue.add(next);
            }
        }
    }
}

深度优先遍历

不使用队列，使用栈。从给定节点开始入栈，然后弹出，当遇到没有注册过的节点，将弹出的节点重新压入，将这个没有注册过的压入，注册之后，重新进行循环

public static void DFS(Node n){
    if(n == null) return;
    Stack<Node> stack = new Stack<>();
    HashSet<Node> set = new HashSet<>();
    stack.push(n);
    set.add(n);
    System.out.println(n.value);
    while(!stack.isEmpty()){
        Node cur = stack.pop();
        for(Node next:cur.nexts){
            if(!set.contains(next)){
                set.add(next);
                stack.push(cur);
                stack.push(next);
                System.out.println(next.value);
                break;
            }
        }
    }
}

拓扑排序

即图中某些点依赖于其他的点，具体到问题中是需要先完成某些事，才能做其他的某些事，所以需要找到最先能够不依赖别的事情的事，完成它们，然后才能完成其他的事，这就形成了一个顺序，即为拓扑排序

前提是没有环，即不存在事件A依赖于事件B，事件B又依赖于事件A

如何解决？可以先找到入度为0的点，然后将从该点和该点指出去的线删掉，找到下一个入度为0的点，依次进行即可

public static List<Node> topologySort(Graph graph){
    // value为节点当前剩余的入度
    HashMap<Node, Integer> map = new HashMap<>();
    // 用于储存入度为0的节点
    Queue<Node> zeroInQueue = new LinkedList<>();

    for(Node each:graph.nodes.values()){
        map.put(each, each.in);
        if(each.in == 0) zeroInQueue.add(each);
    }
    // 至此找到了第一批入度为0的点

    List<Node> result = new ArrayList<>();
    while(!zeroInQueue.isEmpty()){
        Node cur = zeroInQueue.poll();
        result.add(cur);
        for(Node next:cur.nexts){
            int temp = map.get(cur) - 1;
            map.put(next, temp);
            if(temp == 0) zeroInQueue.add(next);
        }
    }
    return result;
}

最小生成树问题

即保证图连通，并且所有边加起来权重最小，下面的两个算法适用于无向图

K (kruskal) 算法

即初始状态下每个点为独立的一个集合，将所有边按权重从小到大排列，然后遍历整个边集，从最小的开始，如果边的from和to不在一个集合中，则合并为一个集合

public static class mySets{
    public HashMap<Node, List<Node>> map;
    public mySets(List<Node> nodes){
        map = new HashMap<>();
        for(Node each:nodes){
            List<Node> set = new ArrayList<>();
            set.add(each);
            map.put(each, set);
        }
    }

    public boolean isSameSet(Node from, Node to){
        List<Node> set1 = map.get(from);
        List<Node> set2 = map.get(to);
        return set1 == set2;
    }

    public void union(Node from, Node to){
        List<Node> set1 = map.get(from);
        List<Node> set2 = map.get(to);
        for(Node each:set2){
            set1.add(each);
            map.put(each, set1);
        }
    }
}

public static class edgeComparator implements Comparator<Edge>{
    @Override
    public int compare(Edge o1, Edge o2) {
        return o1.weight - o2.weight;
    }
}

public static Set<Edge> kruskal(Graph graph){
    mySets myset = new mySets((List<Node>) graph.nodes.values());
    PriorityQueue<Edge> pq = new PriorityQueue<>(new edgeComparator());
    Set<Edge> result = new HashSet<>();
    for(Edge each:graph.edges) pq.add(each);
    while(!pq.isEmpty()){
        Edge edge = pq.poll();
        Node from = edge.from;
        Node to = edge.to;
        if(!myset.isSameSet(from, to)){
            result.add(edge);
            myset.union(from, to);
        }
    }
    return result;
}

可以使用并查集替代mySets，更加高效

p (prim) 算法

该算法不需要突然将两个集合连接起来，具体逻辑为从某点开始，解锁所有与该点连接的边，然后挑选权重最小的边，看该条边另一端的点，是否注册过，没有就加入这条边，直至将所有点遍历完毕

public static Set<Edge> prim(Graph graph){
    Set<Edge> result = new HashSet<>();
    HashSet<Node> set = new HashSet<>();
    PriorityQueue<Edge> pq = new PriorityQueue<>(new edgeComparator());
    // 这个大循环的作用是防止有森林的出现，即整个图不连通，分为几个独立的连通的区域
    for(Node node:graph.nodes.values()){
        if(!set.contains(node)){
            set.add(node);
            for(Edge each:node.edges) pq.add(each);
            while(!pq.isEmpty()) {
                Edge edge = pq.poll();
                Node to = edge.to;
                if (!set.contains(to)) {
                    set.add(to);
                    for(Edge each:to.edges) pq.add(each);
                }
            }
        }
    }
    return result;
}
for(Edge each:to.edges) pq.add(each);

该行代码会重复把同一条边加入堆中，但是并不影响结果，而且边是有限的，外一层每次循环都会弹出一个，不会造成死循环

Dijkstra算法

给定一个出发点，算法会得出从该点到其他点的最小距离，该算法只适用于权重为正数的边

1.0

假定出发点为A，算到B，C，D的最小值

首先定义一张表，含有从出发点到每个点的距离，初始状态当然为到A的为0，到B，C，D的为无穷大

每次选择表中距离最小的点，第一次当然选择A，然后看从该点指出来的边，与表中比对，是否使用这些边使得表中的距离可以变小，然后修改，一轮过后，循环直到遍历所有点（每次循环都会搞定一个点，称之为锁死一个点，之后不会再改变到该点的距离）

public static HashMap<Node, Integer> dijkstra(Node head){
    HashMap<Node, Integer> distanceMap = new HashMap<>();
    HashSet<Node> selectedNodes = new HashSet<>();
    distanceMap.put(head, 0);

    Node minNode = minDistanceUnselectedNode(distanceMap, selectedNodes);
    while(minNode != null){
        int curDistance = distanceMap.get(minNode);
        for(Edge eachEdge:minNode.edges){
            Node toNode = eachEdge.to;
            if(!distanceMap.containsKey(toNode)) distanceMap.put(toNode, curDistance + eachEdge.weight);
            else distanceMap.put(toNode, Math.min(distanceMap.get(toNode), curDistance + eachEdge.weight));
        }
        selectedNodes.add(minNode);
        minNode = minDistanceUnselectedNode(distanceMap, selectedNodes);
    }
    return distanceMap;
}

// 在map中找到没有被锁死的点，而且是距离最小的点
public static Node minDistanceUnselectedNode(HashMap<Node, Integer> map, HashSet<Node> set){
    Node minNode = null;
    int minDistance = Integer.MAX_VALUE;
    for(Map.Entry<Node, Integer> entry:map.entrySet()){
        Node key = entry.getKey();
        int value = entry.getValue();
        if( (!set.contains(key)) && (minDistance > value) ){
            minDistance = value;
            minNode = key;
        }
    }
    return minNode;
}

2.0 堆优化方案

改写小根堆，使堆增加如下性质：

可以修改堆中某个节点的值，经过调整，仍然是小根堆

public static class nodeRecord{
    public Node node;
    public int distance;
    public nodeRecord(Node node, int distance) {
        this.node = node;
        this.distance = distance;
    }
}

public static class nodeHeap{
    private Node[] nodes;
    // 即记录每个节点在上一行nodes数组的下标
    private HashMap<Node, Integer> nodeIndexMap;
    // 记录目前为止到该节点的距离
    private HashMap<Node, Integer> distanceMap;
    private int size;

    public nodeHeap(int size) {
        nodes = new Node[size];
        nodeIndexMap = new HashMap<>();
        distanceMap = new HashMap<>();
        this.size = 0;
    }

    private void heapInsert(int index){
        while(distanceMap.get(nodes[(index - 1) / 2]) < distanceMap.get(nodes[index])){
            swap((index - 1) / 2, index);
            index = (index - 1) / 2;
        }
    }

    private void heapify(int index, int heapSize){
        int left = index * 2 + 1;
        while(left < heapSize){
            int smallest = (left + 1 < heapSize) && (distanceMap.get(nodes[left + 1]) < distanceMap.get(nodes[left])) ? left + 1 : left;
            smallest = distanceMap.get(nodes[smallest]) < distanceMap.get(nodes[index]) ? smallest : index;
            if(smallest == index) break;
            swap(smallest, index);
            index = smallest;
            left = index * 2 + 1;
        }
    }

    private void swap(int i, int j){
        nodeIndexMap.put(nodes[i], j);
        nodeIndexMap.put(nodes[j], i);
        Node temp = nodes[i];
        nodes[i] = nodes[j];
        nodes[j] = temp;
    }

    private boolean isEntered(Node node){
        return nodeIndexMap.containsKey(node);
    }

    private boolean isOnHeap(Node node){
        return isEntered(node) && (nodeIndexMap.get(node) != -1);
    }

    public boolean isEmpty(){
        return size == 0;
    }

    public nodeRecord pop(){
        Node node = nodes[0];
        nodeRecord nr = new nodeRecord(node, distanceMap.get(node));
        nodeIndexMap.put(node, -1);
        distanceMap.remove(node);
        swap(0, --size);
        nodes[size + 1] = null;
        heapify(0, size);
        return nr;
    }

    // 从来没有进入堆过，则add
    // 发现在堆中，但是distance更小，则update
    // 发现在堆中，但是并没有更小，则ignore
    public void addOrUpdateOrIgnore(Node node, int distance){
        if(!isEntered(node)){
            nodes[size] = node;
            nodeIndexMap.put(node, size);
            distanceMap.put(node, distance);
            heapInsert(size++);
        }
        if(isOnHeap(node)){
            distanceMap.put(node, Math.min(distanceMap.get(node), distance));
            heapInsert(nodeIndexMap.get(node));
        }
    }
}

public static HashMap<Node, Integer> dijkstra2(Node head, int size){
    nodeHeap heap = new nodeHeap(size);
    HashMap<Node, Integer> res = new HashMap<>();
    heap.addOrUpdateOrIgnore(head, 0);
    while(!heap.isEmpty()){
        nodeRecord record = heap.pop();
        Node cur = record.node;
        int distance = record.distance;
        for(Edge edge:cur.edges)
            heap.addOrUpdateOrIgnore(edge.to, distance + edge.weight);
        res.put(cur, distance);
    }
    return res;
}
